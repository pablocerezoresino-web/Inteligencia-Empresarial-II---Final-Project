### B) War Game Design — **1 página del informe**
**Red/Blue/Green Team** (TN02):

- **Red Team** = 50 Doctors (atacante)
- **Blue Team** = Sanitas + Adeslas (defensores)
- **Green Team / Control** = Regulador / Mercado / Pacientes

Definir:
- Roles y número de jugadores por equipo
- Information packages: qué información recibe cada equipo
- Rondas: estructura temporal (ej. 3 rondas de 30 min)
- **2+ Shock Events**: eventos sorpresa que se introducen durante el juego
  - Ejemplo: "Nueva regulación DGS limita inversión extranjera en sanidad"
  - Ejemplo: "Sanitas anuncia alianza con hospital mexicano rival"
